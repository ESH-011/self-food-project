## Project Description
This is an academic C++ project for managing a self food menu.

This program shows a menu and gets user input.
This program has 3 menus :
1. Login Menu      
2. Admin Menu     
3. Student Menu

In Logn Menu: Users can choose to be an administrator or a student.
After choosing, users must type the correct username and password to enter.

In Admin Menu, Administrator can:
1. Add/delete/edit info students + list of students     
2. Add/delete/edit info food  + list of available food + reservations of the day   
3. See reports of total income and refunds + most reserved food + students with most reserved     
4. change their own username and password 

In Student Menu, Students can:
1. view food menu    
2. reserved food    
3. cancel their reservations    
4. see their reservations    
5. add money to their wallet    
6. change their own password

*This program can accept both normal and Hash passwords.*

*Administrator first username and password is "admin".*
